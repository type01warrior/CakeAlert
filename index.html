Firestore Security Rules — Required Step

If sync shows Permission denied, you need to paste these rules in your Firebase Console:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
  1. Go to Firebase Console → Firestore → Rules
  2. Replace everything with the rules above
  3. Click Publish
  4. Come back and tap Sync now

Also confirm you are signed in with the same email on both devices.
